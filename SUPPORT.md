# 🧭 Support

Use GitHub issues for reproducible bugs and focused feature requests.

🔎 Before opening an issue:

- Confirm Home Assistant can read the local file or fetch the remote URL.
- Confirm the JSON is valid.
- Remove private account data, tokens, usernames, and local paths from logs.
- Check `README.md` for current limitations.

📋 Please include:

- HA Idleon version.
- Home Assistant version.
- Data source type: `local_file` or `remote_url`.
- The relevant error message.
- A sanitized sample JSON shape if the parser is failing.

🔐 Do not include real account exports or private URLs.
