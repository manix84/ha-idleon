# 🛡️ Security Policy

## ✅ Supported Versions

Only the latest released version of HA Idleon is supported.

## 🚨 Reporting A Vulnerability

Please report security issues privately through GitHub security advisories when
available. If that is not available, open a minimal issue that does not include
private data and ask for a private contact path.

🔐 Do not publish:

- Idleon account exports.
- Private remote URLs.
- Session tokens.
- Browser cookies.
- Home Assistant secrets.
- Local file paths that expose usernames or infrastructure.

## 🚧 Project Boundaries

HA Idleon does not implement login, credential collection, browser scraping,
session scraping, token scraping, or write actions. Reports suggesting those
features will be treated as out of scope for v1.
